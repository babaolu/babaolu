### [TerAI](https://github.com/babaolu/TerAI)
A native ARM64 rewrite of the TerAI system, engineered for zero-latency startup and minimal interpreter overhead by compiling directly with `clang++` in Termux. This project focuses on efficient on-device AI execution, incorporating a direct in-process `libllama.so` linkage for local LLM inference, advanced tab-completion, and exposing tool schemas for enhanced functionality and user interaction.
Stack: C++, CMake, Shell, libcurl, nlohmann/json, readline, libllama.so

### [MyG](https://github.com/babaolu/MyG)
Known as VulkanMind, this local multi-agent AI system is tailored for modern Vulkan graphics programming and high-performance C++ systems engineering. It intelligently orchestrates specialized agents for tasks such as platform detection, knowledge retrieval, Vulkan-Hpp/VMA code generation, validation, and debugging, including a self-improvement loop for continuous refinement and adaptation to complex engineering challenges.
Stack: Python, FastAPI, LangGraph, Qdrant, SQLite, Pydantic, Vulkan-Hpp/VMA

### [invitation](https://github.com/babaolu/invitation)
A robust web application designed for event management, capable of generating unique, permanent invitation pages and live PDFs for each guest. Its innovative design ensures that if guest or table details change, the content on their personalized link and PDF updates instantly, eliminating the need for re-sending. It includes an admin dashboard for guest management and responsive seating directory features for efficient event coordination.
Stack: JavaScript, HTML, PLpgSQL, Supabase (PostgreSQL), Vite, React

### [Poot](https://github.com/babaolu/Poot)
A decentralized physical infrastructure network (DePIN) that connects mobile device miners with cloud customers through a demand-gated token economy. This platform ensures tokens are minted only when real customer instances consume miner resources, backing the native token with economic utility. It features decentralized zero-knowledge storage using erasure coding (Reed-Solomon) and AES-256-GCM encryption for data privacy and K-redundancy, recently passing testnet.
Stack: C++, Python, TypeScript, CMake, JavaScript, Docker, PostgreSQL, Reed-Solomon, AES-256-GCM
